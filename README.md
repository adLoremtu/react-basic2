## ディレクトリ構成

- react-basic
  - 勉強会で使用するサンプルディレクトリです。
  - こちらを使って色々実装します。
- react-basic-answer
  - 勉強会の回答を記載したディレクトリです。
  - 手っ取り早く答えを知りたい方はこちらを見ていただければと思います。

## react 起動方法

`cd react-basic`<br>
`npm run start`

localhost:3000 が立ち上がり、画面が表示されれば OK です。
